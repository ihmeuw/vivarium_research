===================
MACE RCT Simulation
===================

.. contents::
   :local:

Overview
++++++++

This concept model describes a simulation of a randomized controlled trial
(RCT) designed to reduce major adverse cardiovascular events (MACE) by
lowering low-density lipoprotein cholesterol (LDL-C) using an experimental
new agent.

Details of the trial design, population, interventions, and outcomes will
be added in future revisions of this document.
